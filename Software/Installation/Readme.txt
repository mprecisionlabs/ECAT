The file 'ECAT Software Installation.wse' is the script used
to create the ECAT SurgeWare/BurstWare/PQFWare installation diskette(s)/CD 
with the Wise Installation System (from Wise Solutions, Inc.), v8.0 or higher.  
The installation media created can be used to install these applications on 
Windows 9x/Me/2000/XP.
